# Security Policy

## Reporting a vulnerability

If you believe you have found a security issue in Seller Block Marketplace, do not open a public issue first.

- GitHub advisories: https://github.com/LegionofMany/Seller-Block-marketplace/security/advisories/new
- Repository: https://github.com/LegionofMany/Seller-Block-marketplace

Include the affected area, reproduction steps, impact, and any proposed mitigation.

## Verification surfaces

This repository may be referenced from the live marketplace site, the project wiki, and security contact files as part of the public trust and verification layer for the product.
